วิธีเชื่อมกับ Google Sheets:
1. เปิด Google Sheets ที่มีไฟล์ Excel นี้
2. ไปที่ Extensions → Apps Script
3. วางโค้ด Apps Script ที่ได้มาพร้อมไฟล์นี้ (ไฟล์ AppScript.gs)
4. กด Deploy → New deployment → Web App → Execute as Me → Anyone
5. Copy URL ที่ได้แล้ววางด้านล่างนี้

ถ้าเปิดผ่าน Google Sheets (iframe) ระบบจะดึงข้อมูลอัตโนมัติผ่าน google.script.run
